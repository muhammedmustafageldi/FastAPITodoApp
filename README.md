## Installation

1. Clone the project:
git clone https://github.com/muhammedmustafageldi/FastAPITodoApp.git
2. Install required packages:
pip install -r requirements.txt
3. Set up environment variables:
cp .env.example .env
4. Start the application:
uvicorn main:app --reload
